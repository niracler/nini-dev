# Change: 新增 Codex vs Claude 对比附录页

## Why

现有的「AI 编程工具：三代范式」页面提供高层次的演进概览，适合正文。用户调研挖掘了丰富的社区情绪数据（500+ Reddit 评论、HN 讨论）和两大主流 CLI Agent（Codex CLI vs Claude Code）的行为洞察，这些深度对比内容更适合作为**附录**供感兴趣的听众深入了解。

## What Changes

- **保留**：「AI 编程工具：三代范式」页面（原样不动）
- **新增**：「Codex CLI vs Claude Code」对比页面作为**附录**，包含：
  - 性能对比表（代码质量、上下文窗口、执行速度、交互风格）
  - 社区偏好数据（65.3% vs 34.7%，含加权分析）
  - 行为特征对比（指令遵循方式、执行反馈风格）
  - HN「精灵」引用（增强记忆点）
  - 场景适配建议
  - 核心结论：「没有银弹」
  - 详细脚注（6+ 来源）

## Impact

- 影响的 spec：`slidev-presentation`
- 影响的代码：`repos/ai-programming-share/slides.md`
- 页数变化：+1（新增附录页）
- 位置：「资源分享」页面之前（约第 2371 行）
- 视觉锚点：对比表格作为信息型视觉元素，符合 spec 要求
