## 附录: 评分公式

```
Score = 0.3×频率 + 0.25×复杂度 + 0.25×可复用性 + 0.2×组合性
```

每个因素归一化到 0-1 范围:

- **频率**: session_count / max_session_count
- **复杂度**: 主观 1-5 → 0.2-1.0
- **可复用性**: 跨项目适用性 (低=0.4, 中=0.6, 高=0.9)
- **组合性**: 集成潜力 (低=0.4, 中=0.6, 高=0.9)
